Rules of The Game

  • The computer picks a sequence of colors. The number of colors is the code length.
  • The objective is to guess the exact positions of the colors.
  • By default, each color can only be used once. If you enable "Allow Duplicates," colors may repeat.
  • Red peg: correct color & correct position
  • White peg: correct color but wrong position
  • Black peg: color not in code
  • You have 10 attempts. Good luck!

How to play the game

  • Start a new game by clicking on the 'Start Game' button. If you wish to change the default game parameters, you may change the 'Allow duplicates' field before clicking on the 'Start Game' button.
  • Select a color at the bottom and place it in the current guess line above by clicking on the desired position.
  • Click 'Check' to get feedback after your guess.
  • You have 10 attempts to guess the code sequence. Good luck!
